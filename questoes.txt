1.O que é um repositório no Git e qual a sua importância?
O repositório git é sua pasta principal que armazena o histórico de informações e alterações do projeto, ele é importante pelo sua segurança e organização.

2.Qual a função do Git Bash?
é um terminal de comando do proprio do git.

3. que faz o comando git add e qual a diferença entre git add <arquivo>e git add .?
git add é para adicionar alterações do arquivo, git add <arquivo> adiciona somente o arquivo mencionado, gite add . inclui todos arquivos e adiciona as alterações.

4.Por que usamos o ponto ( .) no comando git add .?
o . é referencia do diretório atual que voce está localizado, entao usamos ele para adicionar o diretório todo.

5.Explique o comando git commite por que é importante criar uma mensagem descritiva.
Para saber oque voce alterou no seu projeto, ter organização e segurança caso faça alguma alteraçao no projeto que não funcione.

6.Qual a função do comando git push?
a função dele é mandar os commites para seu repositório 

7. Por que fazer commits frequentes?
Ajuda a manter tudo organizado, seguro e fácil de acompanhar no projeto.

8. O que são tags HTML?
São os blocos que montam a página, como títulos, parágrafos, imagens, etc.

9. O que são atributos em uma tag HTML?
São detalhes extras que mudam o comportamento da tag (como links, tamanhos, descrições).

10. O que é o Emmet?
É uma ferramenta que acelera o código com atalhos. Digita menos, produz mais.

11. O que é box-sizing?
Controla como o tamanho dos elementos é calculado. Usamos no reset para tudo ficar padrão.

12. Por que usar tags semânticas?
Porque deixam o código mais claro e ajudam na acessibilidade e no Google.

13. O que são media queries?
São regras no CSS para deixar o site bonito em qualquer tamanho de tela.

14. O que é o GitHub Pages?
Um jeito fácil e grátis de colocar seu site no ar usando o GitHub.

15. O que é Flexbox?
Uma forma fácil de alinhar e organizar elementos no layout com CSS.

16. Justify-content vs Align-items:

justify-content: alinha na direção principal (horizontal ou vertical).

align-items: alinha na outra direção (cruzada).

17. O que faz o flex-grow?
Faz um item crescer para ocupar mais espaço no contêiner Flexbox.

18. Diferença entre column e row no Flexbox:

row: itens na horizontal.

column: itens na vertical.
