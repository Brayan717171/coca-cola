1.O que é um repositório no Git e qual a sua importância?
O repositório git é sua pasta principal que armazena o histórico de informações e alterações do projeto, ele é importante pelo sua segurança e organização.

2.Qual a função do Git Bash?
é um terminal de comando do proprio do git.

3. que faz o comando git add e qual a diferença entre git add <arquivo>e git add .?
git add é para adicionar alterações do arquivo, git add <arquivo> adiciona somente o arquivo mencionado, gite add . inclui todos arquivos e adiciona as alterações.

4.Por que usamos o ponto ( .) no comando git add .?
o . é referencia do diretório atual que voce está localizado, entao usamos ele para adicionar o diretório todo.

5.Explique o comando git commite por que é importante criar uma mensagem descritiva.
Para saber oque voce alterou no seu projeto, ter organização e segurança caso faça alguma alteraçao no projeto que não funcione.

6.Qual a função do comando git push?
a função dele é mandar os commites para seu repositório 

7.Qual a importância de criar compromissos frequentes durante o desenvolvimento de um projeto?

8.Explique a função das tags HTML e como elas estruturam uma página.
As tags HTML são elementos que definem a estrutura e o conteúdo de uma página web. Cada tag representa um tipo de conteúdo (como texto, imagem, link, título, parágrafo) e ajuda a organizar e formatar a página Elas formam uma estrutura em árvore, facilitando o entendimento do navegador e o estilo via CSS.

9.O que são atributos em uma tag HTML e qual a sua função?
Os atributos são informações adicionais dentro das tags HTML que modificam ou configuram o comportamento do elemento.O atributo href define para onde o link aponta. Outros exemplos comuns são class, id, src, alt, etc.

10.O que é o Emmet e como ele facilita o desenvolvimento de páginas HTML?
Emmet é um plugin que permite escrever abreviações que são automaticamente expandidas para código HTML ou CSS completo, Ele acelera muito o processo de codificação.

11.O que é o box-sizing no CSS e por que usamos ele no reset inicial?
O box-sizing define como as dimensões de um elemento são calculadas.

12.Por que é importante usar tags semânticas no HTML?
As tags semânticas (como <header>, <main>, <article>, <footer>) dão significado ao conteúdo da página, tornando o HTML mais claro.

13.O que são media queries no CSS e qual a sua função?
As media queries são regras CSS que aplicam estilos condicionalmente, dependendo do tamanho da tela, resolução, orientação, etc.

14.Qual a função do GitHub Pages?
O GitHub Pages permite hospedar sites estáticos (HTML, CSS, JS) diretamente de um repositório do GitHub, de forma gratuita. É muito usado para portfólios, documentação e projetos simples.

15.O que é o Flexbox no CSS e para que ele é utilizado?
O Flexbox é um modelo de layout CSS que facilita o alinhamento e distribuição de espaço entre elementos dentro de um container. Ele é muito útil para criar layouts flexíveis, responsivos e centralizados.

16.Qual a diferença entre justify-contente align-item sem um container Flexbox?
justify-content: alinha os itens horizontalmente (eixo principal).
align-items: alinha os itens verticalmente (eixo cruzado).

17.Como flex-growfunciona e qual seu propósito no layout Flexbox?
O flex-grow define a proporção de crescimento de um item em relação aos demais dentro de um container Flex.

18.Explique a diferença entre flex-direction: column;e flex-direction: row;.
flex-direction: row; (padrão): os itens são posicionados na horizontal (da esquerda para a direita).
flex-direction: column;: os itens são posicionados na vertical (de cima para baixo).
